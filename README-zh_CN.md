# 问卷编辑器
- 一个问卷调查编辑器。包含管理端自定义问卷，用户侧提交答案
- 管理端技术栈：React18 + React-Router-Dom + redux + TS4 + antd5
- 用户端技术栈：Nextjs + Redux + axios
- 后端数据层技术栈：pm2 + nodejs + koa + koa-route
